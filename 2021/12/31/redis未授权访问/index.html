目标 IP：192.168.1.16

# 靶场搭建

在 centos7 使用 [vulhub]{.kbd .red} 搭建 redis 未授权访问漏洞环境

# 复现步骤

使用 nmap 查询信息

显示脚本运行错误

根据大佬的解决方法修改 redis-info.nes 脚本第 139 行

for _, item in ipairs(restab) do
	local ip = item:match("addr=%[?([0-9a-f:.]+)%]?:[0-9]+ ")
	if not item or 0 == #item then 
		break
	end
	client_ips[ip] = true;
end

脚本运行成功

使用 redis-cli 连接目标机器的 redis 数据库
